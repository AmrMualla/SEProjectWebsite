    We aim to develop a user-friendly web application to simulate 1D Gel Electrophoresis, aligning with current software engineering standards. The goal is to create a versatile shell to host this simulation and enable the seamless integration of future simulations by subsequent teams. The project includes a transformation of the existing Java code base into a more accessible JavaScript/Python framework, ensuring adaptability and broader utility. Comprehensive, user-friendly instructions will be provided, allowing individuals without a computing background to easily update and refactor the project. In addition to retaining existing features, new functionalities will be incorporated for more precise identification and analysis of specific proteins. This initiative combines technological advancement with inclusivity, aiming to deliver a versatile and future-proof simulation platform in the scientific domain.
